### 기능

macbook 의 메모장 앱을 구현한 웹 어플리케이션입니다.

- 주요기능

  - 로그인을 통해서 자기 메모만을 볼수 있는 기능
  - CRUD 기능 구현 (통신)
  - 리스트 만들기
  - 화면 나누기

- 추가기능

  - 저장버튼 없애기
  - 잠금기능
  - 이미지 넣기
  - 메모 검색(제목과내용)
  - SNS 공유기능

- Page
  - Login page, memo page, register page
